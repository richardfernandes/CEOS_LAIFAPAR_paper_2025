# CEOS_LAIFAPAR_paper_2025
Documents, data and results for CEOS LAI FAPAR validation paper 2025


This repo contains material for the CEOS WGCV Biophysical Group paper "V10 A Methodology for Validating Synoptic Decametric Resolution Satellite Leaf Area Index and fAPAR Products."

Folders (not all are populated as of November 15, 2025):

manuscript - materials for submission and review
documents - supporting documents, mainly major references
materials - materials identified in manuscript
methods - methods identified in manuscript
CEOS - ceos specific items
